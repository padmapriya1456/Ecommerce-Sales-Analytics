# E-Commerce Sales Performance Analytics

This project analyzes e-commerce sales data using Microsoft Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Dashboarding

## Analysis Performed
- Monthly Sales Trend
- Top Selling Products
- Country-wise Sales
- Customer Analysis

## Dashboard Features
- KPI Cards
- Interactive Charts
- Business Insights

## Outcome
Created a business-ready dashboard to analyze sales performance and generate insights.
